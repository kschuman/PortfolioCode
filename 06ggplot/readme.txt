This plot shows the change in number of murders from 2014 to 2016. The light blue dot shows the number of murders that occured in 2014 for each city, and the dark blue dot is for 2016. We can see that overall, there weren't huge changes in the number of murders for most cities, and we can also easily see which cities have the most murders. Unsuprisingly, the larger cities are at the top of this list.
