This heatmap shows the similarities between the word 'mother' in many Indo-European languages. Each word's phonetic spelling was matched against each other word, which created the similarity score (0 to 1). For example, the Sanstrik work 'matr' is phonetically the same as the Serbian work 'mater'. Since all these languages stem from one original language, it's not suprising that many of the words have similar sounds.
